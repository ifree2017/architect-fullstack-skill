# Web 前端任务示例

适用于 Web 管理后台、ToB 平台、ToC H5、官网、活动页、SSR/CSR 页面等场景。目标是让 AI 在 Web 场景下一开始就进入正确上下文，而不是只给泛前端建议。

---
